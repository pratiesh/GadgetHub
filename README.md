# GadgetHub
GadgetHub is a comprehensive e-commerce platform designed for selling a wide range of electronic gadgets. The website offers a seamless shopping experience with features such as product browsing, user authentication, shopping cart management, and secure checkout. Utilizing Advanced Java and JDBC, the project ensures efficient database interactions and dynamic content management. My contributions included developing the backend logic, integrating database operations, and implementing user-friendly UI/UX elements. This project demonstrates my proficiency in full-stack development and my ability to create functional, user-oriented web applications.
